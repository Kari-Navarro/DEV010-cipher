Generador de contraseñas cifrado Cesar

Crea constraseñas seguras para proteger tus cuentas basado en el método del cifrado Cesar🔒. Ingresa una contraseña a cifrar en letras mayúsculas, agrega un número para obtener el desplazamiento y obten el resultado👩‍💻. Esta página NO permite cifrar con letras minúsculas👀 o cualquier otro signo 😪. El desplazamiento se obtendrá números enteros 👍.
